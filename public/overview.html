Github link: https://github.com/pmservice/profits-forecast

This sample application uses two versions of the catalog_streaming_timeseries.str SPSS Modeler stream and training data from the SPSS Modeler Demos. It adopts Watson Machine Learning service capabilities (Bluemix platform) to predict the sales of men's and women's clothing.

Within this sample application you are able to:

  • provide sales input data for either women's or men's clothing using web application form
  • run time series model by pressing "Score Now" button
  • display information about predicted profit in form of table
